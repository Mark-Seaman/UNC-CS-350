<h3 id="how-to-set-up-django-hosting-at-python-anywhere">
How to set up Django Hosting at Python Anywhere
</h3>
<ul>

<a href="https://www.youtube.com/watch?v=Y4c4ickks2A">Watch video tutorial</a> (7 steps)
</li>
</ul>
<h3 id="learn-linux-commands">
Learn Linux Commands
</h3>
<ul>
<li>
Master a few command line basics
</li>
<li>
Read <a href="CommandLine.md">Command Line</a>
</li>
<li>
Essential commands: pwd, cs, ls, rm
</li>
</ul>
<h3 id="run-terminal-window">
Run Terminal Window
</h3>
<ul>
<li>
Select &quot;Consoles, New console, Bash&quot;
<ul>
<li>
pwd
</li>
<li>
cd
</li>
<li>
ls
</li>
</ul>
</li>
</ul>
<h3 id="clone-your-git-repository">
Clone your git repository
</h3>
<ul>
<li>
Visit your <a href="https://github.com/Mark-Seaman/Book-Builder.git">Github Repo</a>
</li>
<li>
<p>
Clone the repo
</p>
<p>
git clone https://github.com/Mark-Seaman/Book-Builder.git
</p>
</li>
</ul>
<h3 id="create-virtual-environment">
Create Virtual Environment
</h3>
<ul>
<li>
<p>
Create an isolate python environment
</p>
<p>
mkvirtualenv --python=/usr/bin/python3.8 .venv
</p>
</li>
<li>
<p>
Activate the environment
</p>
<p>
source .venv/bin/activate
</p>
</li>
<li>
<p>
Install Django in the environment
</p>
<p>
pip install django
</p>
</li>
</ul>
<h2 id="create-web-app">
Create Web App
</h2>
<ul>
<li>
Visit your main page
</li>
<li>
Login to <a href="https://www.pythonanywhere.com">Python Anywhere account</a>
</li>
<li>
Bookmark this page on your bookmark toolbar
</li>
<li>
Select Web tab in top menu
</li>
</ul>
<h3 id="configure-your-web-app">
Configure Your Web App
</h3>
<ul>
<li>
Your Python Anywhere account support one free Web App
</li>
<li>
Free accounts must be enabled every three months
</li>
<li>
Click on &quot;Run until 3 months from today&quot; button
</li>
<li>
This is not a problem for when you are actively developing code
</li>
</ul>
