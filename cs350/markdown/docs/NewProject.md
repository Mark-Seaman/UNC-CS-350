            &lt;h1 id=&quot;new-book-builder-project&quot;&gt;New Book Builder Project&lt;/h1&gt;</code></pre>
<p>
Steps to create a new app from scratch
</p>
<h3 id="setup-the-environment">
Setup the environment
</h3>
<pre><code>python3 -m venv .venv

source .venv/bin/activate

pip freeze  # Includes Django

which python</code></pre>
<h3 id="create-a-blank-django-project">
Create a blank Django project
</h3>
<pre><code>django-admin startproject bookbuilder

tree bookbuilder

bookbuilder
├── bookbuilder
│   ├── __init__.py
│   ├── asgi.py
│   ├── settings.py
│   ├── urls.py
│   └── wsgi.py
└── manage.py</code></pre>
<h3 id="test-the-blank-project">
Test the blank project
</h3>
<pre><code>cd bookbuilder

python manage.py migrate

python manage.py runserver

Browse http://127.0.0.1:8000/</code></pre>
<h3 id="add-book-app">
Add &quot;book&quot; app
</h3>
<pre><code>python manage.py startapp book

tree 

.
├── book
│   ├── __init__.py
│   ├── admin.py
│   ├── apps.py
│   ├── migrations
│   │   └── __init__.py
│   ├── models.py
│   ├── tests.py
│   └── views.py
├── bookbuilder
│   ├── __init__.py
│   ├── __pycache__
│   ├── asgi.py
│   ├── settings.py
│   ├── urls.py
│   └── wsgi.py
├── db.sqlite3
└── manage.py</code></pre>
<p>
Create HTML templates
</p>
<pre><code>mkdir templates</code></pre>
<p>
Edit app2/settings.py to load the templates
</p>
<pre><code>TEMPLATES = [
    {
        ...
        &#39;DIRS&#39;: [&#39;templates&#39;],
        ...
    },
]</code></pre>
<p>
Create templates/index.html
</p>
<pre><code>&lt;!DOCTYPE html&gt;
&lt;html lang=&quot;en&quot;&gt;

&lt;head&gt;
    &lt;meta charset=&quot;UTF-8&quot;&gt;
    &lt;title&gt;Book Builder&lt;/title&gt;
&lt;/head&gt;

&lt;body&gt;
    &lt;h1&gt;Book Builder&lt;/h1&gt;
&lt;/body&gt;

&lt;/html&gt;</code></pre>
<p>
Edit bookbuilder/urls.py
</p>
<pre><code>from django.views.generic import TemplateView
from django.urls import path


urlpatterns = [
    path(&#39;&#39;, TemplateView.as_view(template_name=&#39;index.html&#39;)),
]</code></pre>
<p>
Test the application
</p>
<pre><code>python manage.py runserver

browse to http://127.0.0.1:8000/</code></pre>
